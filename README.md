# dotfiles

```
dotfiles/
├── laptop/   → Lenovo ThinkPad (Linux Mint 22.3)
└── desktop/  → Main PC (coming soon)
```

## laptop (HP Pavilion dv7 — Linux Mint 22.3)

- **WM:** niri (Wayland compositor)
- **Bar:** waybar (Catppuccin Mocha)
- **Launcher:** fuzzel
- **Notifications:** dunst
- **Terminal:** alacritty
- **Wallpaper:** `~/Downloads/storm.jpg`

Deploy with stow:

```sh
stow -t ~ laptop
```
